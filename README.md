# Usage
<!-- usage -->
```sh-session
$ npm install -g minicomp
$ minicomp COMMAND
running command...
$ minicomp (-v|--version|version)
minicomp/0.0.0 darwin-x64 node-v10.6.0
$ minicomp --help [COMMAND]
USAGE
  $ minicomp COMMAND
...
```
